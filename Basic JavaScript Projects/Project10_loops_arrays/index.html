The Let Keyword

The let keyword declares a block scope local variable, optionally initializing it to a value. In this example, the variable X is declared with the let keyword inside a block. The value of X inside the block is different from the value of X outside the block.

The return statement stops the execution of a function and returns a value from that function. In this example, the function return_function returns a greeting message that includes the name stored in the variable name. The returned message is displayed in the paragraph with the id Return when the button is clicked.
